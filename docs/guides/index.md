# Guides

Detailed guides for working with OpenRV Web.

## Planned Topics

- **Color Management** -- Working with LUTs, CDL, OCIO configs, and the display pipeline
- **Media Formats** -- Loading EXR, DPX, HDR, and video files
- **Scripting API** -- Automating workflows with `window.openrv`
- **Plugin Development** -- Creating custom exporters and UI panels
- **Keyboard Shortcuts** -- Full reference for all keyboard shortcuts
- **Annotations** -- Using paint and text annotation tools
- **Scopes & Analysis** -- Histogram, waveform, vectorscope, and pixel probe
